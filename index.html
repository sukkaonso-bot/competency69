การประเมินสมรรถนะพยาบาลวิชาชีพ ปี 2569

ศูนย์รวมรายงานและตัวชี้วัดประสิทธิภาพของบุคลากรทางการพยาบาล กลุ่มภารกิจด้านการพยาบาล ออกแบบมาเพื่อให้ผู้บริหาร หัวหน้ากลุ่มงาน และผู้มีส่วนเกี่ยวข้อง สามารถวิเคราะห์และเข้าถึงผลคะแนนได้อย่างรวดเร็ว ถูกต้อง แม่นยำ และมีความเป็นระเบียบในระบบคลังความรู้เดียวกัน บนเครือข่ายความปลอดภัยของหน่วยงาน

คำแนะนำสำคัญ: วิธีแก้ไขและป้องกัน Error 404 เมื่อกดสลับหน้าเว็บ

หากคุณคลิกเลือกที่เมนูด้านซ้ายแล้วเจอหน้าจอขาวพร้อมคำว่า "404 Not Found" เป็นเพราะชื่อไฟล์ที่อัปโหลดขึ้น GitHub ไม่ตรงกับที่ระบบเรียกใช้ในรหัสโค้ด วิธีตรวจสอบและแก้ไขทำได้ตามขั้นตอนนี้ครับ:

1. ตรวจสอบชื่อไฟล์ในคลังข้อมูล (Repository)

ให้แน่ใจว่าไฟล์ที่อัปโหลดเข้า GitHub มีชื่อเขียนสะกดตรงกับด้านล่างนี้ทุกตัวอักษร:

  • index.html (หน้าเว็บหลักนี้)
  • dashboard_2569 (2).html
  • ประเมินสมรรถนะพยาบาล_ปี2569_อัปเดตระบบแล้ว (1).html
  • nurse_competency_dashboard_ready.html
2. วิธีการอัปโหลดที่ง่ายที่สุด

ไม่ต้องเปลี่ยนชื่อไฟล์เดิมของคุณในคอมพิวเตอร์เลย! ให้เปิดหน้าเว็บ GitHub แล้วกด Add file -> Upload files จากนั้นลากไฟล์ทั้ง 3 ไฟล์นั้นมาวางได้ทันที ระบบจะอัปโหลดชื่อนั้นให้อัตโนมัติและสลับเปลี่ยนหน้าได้ทันที!

รายงานการประเมินหัวหน้าหน่วยงาน

สรุปผลคะแนนสมรรถนะจริงของหัวหน้าหน่วยงานในแต่ละแผนก/หอผู้ป่วย มีข้อมูลสถิติแยกตามประสบการณ์ การประเมินผ่านเกณฑ์ และสรุปข้อควรพัฒนาอย่างเด่นชัด

สรุปการประเมินสมรรถนะการปฏิบัติตามหน้าที่ 2569

เครื่องมือจัดสรร นำเข้าข้อมูล และคำนวณสถิติเพื่อแบ่งการประเมิน 3 มิติอย่างสมบูรณ์แบบ พร้อมระบบนำเข้าไฟล์ Excel เพื่ออัปเดตข้อมูลพนักงาน

สรุปการประเมินสมรรถนะหลัก 2569
